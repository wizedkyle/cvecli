# cvecli reset-secret

Resets the secret for a user in the organization

The user account that is being used to authenticate needs to have the ADMIN role to perform this action.

```shell
cvecli reset-secret [flags]
```

## Options

```
-h, --help - help for reset-secret
-u, --username - Specify the username which needs the secret reset
```
